# CHRONOLOGIE — Luxury Watches

A curated marketplace for luxury timepieces, built with **Next.js 16**, **TypeScript**, **Tailwind CSS v4**, and **GSAP** for animations.

## ✨ Features

- **Hero Section** — Animated title with countdown timer to next event
- **On Tour** — Global event schedule with location cards and mini countdowns
- **Marquee** — Infinite scrolling text banner with GSAP scroll-linked animation
- **About** — Event description with reveal-on-scroll animations
- **Ticket Benefits** — Grid of benefits with hover effects
- **Past Lineup** — Brand showcase grid
- **Gallery** — Masonry-style photo grid with hover overlays
- **FAQ** — Accordion with smooth open/close animations
- **Footer** — Social links with legal notice

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
